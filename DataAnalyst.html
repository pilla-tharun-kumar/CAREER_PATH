PHASE 1 Excel + Statistics + SQL Basics Month 1–2 · Weeks 1–8
📊 This is your foundation. Don't rush to Python or Tableau yet — recruiters still test Excel and basic SQL in 90% of fresher data analyst interviews. Nail this first.
MON–TUE · Core Excel Functions
Learn VLOOKUP, HLOOKUP, INDEX-MATCH — the most common interview Excel questions
→ YouTube Free
Learn IF, SUMIF, COUNTIF, AVERAGEIF — conditional logic used daily by analysts
→ Microsoft Docs
Learn TEXT functions: TRIM, LEFT, RIGHT, MID, CONCATENATE — essential for dirty data cleaning
Practice with a real dataset — download a free retail sales dataset from Kaggle and apply all functions
→ Kaggle Dataset
WED–THU · Pivot Tables & Charts
Master Pivot Tables — summarize, group, filter, sort large datasets in minutes
→ YouTube Free
Create 3 chart types: Bar chart, Line chart, Pie chart — using the retail dataset
Learn Conditional Formatting — highlight cells, data bars, color scales
FRI · Review + Mini Project
Mini Project: Download the Superstore dataset and create a 1-page Excel dashboard — sales by region, top 5 products, monthly trend
→ Superstore Dataset
Upload your Excel file to GitHub — create a repo "excel-projects" with a README describing what you built

PHASE 2 Python for Data Analysis + Advanced SQL Month 3–4 · Weeks 9–16
🐍 This is where you separate yourself from other freshers. Most candidates know Excel and basic SQL — very few can write clean Python for analysis. This phase makes you stand out.
WEEK 9 · Python Core Concepts
Learn Python basics: Variables, data types, lists, dicts, loops, functions — analyst-focused approach
→ Kaggle Python (Free)
Set up Jupyter Notebook — run your first Python analysis in a notebook
→ Try Jupyter Online
Complete CS50P (Python) first 3 lectures — best free Python course by Harvard
→ CS50P Free
Do NOT spend time on OOP or decorators yet — focus on data manipulation-relevant Python
WEEK 10 · Pandas Deep Dive
Learn Pandas: read_csv, head/tail, info, describe, value_counts, groupby, merge, fillna, dropna
→ Kaggle Pandas (Free)
Replicate your Excel analysis in Python — take the Superstore dataset and answer the same questions using Pandas
Practice: Solve 5 Kaggle Python problems per day in a notebook
LinkedIn: Post "Just learned Pandas — here's how Python replaced 3 hours of Excel work in 10 lines of code" with a code screenshot
WEEK 11 · Matplotlib Basics
Learn Matplotlib: line plots, bar charts, scatter plots, histograms, subplots
→ YouTube Free
Learn Seaborn: heatmaps, pairplots, boxplots, violin plots — beautiful statistical charts in 3 lines
→ Official Docs
Rule: Every chart must have a title, axis labels, and a 1-line insight — no unlabeled charts in your portfolio
WEEK 12 · EDA Project
Portfolio Project 3: Full Exploratory Data Analysis (EDA) on a Kaggle dataset of your choice — 10+ visualizations, 5 key insights, clean Jupyter notebook
→ Movie Dataset (example)
Structure your notebook: Problem statement → Data overview → Cleaning → Exploration → Insights → Recommendations
Push to GitHub + publish on Kaggle — Kaggle notebooks get views and upvotes, building your public profile
LinkedIn Post: Share your EDA notebook link — "Analyzed 5,000 movies. Here are the 3 most surprising insights I found..."
WEEK 13 · Business-Focused SQL Patterns
Learn Cohort Analysis in SQL — most common analyst query pattern in e-commerce, SaaS
→ YouTube Tutorial
Learn Funnel Analysis in SQL — tracking user conversion steps
Practice on real company-style datasets via StrataScratch — sorted by company (Google, Amazon, Meta)
→ StrataScratch Free
WEEK 14 · SQL in Python (SQLAlchemy + Pandas)
Learn to run SQL inside Python notebooks using pandas.read_sql() and SQLite — connects your two biggest skills
Practice on Google BigQuery sandbox — write SQL on real large datasets (free)
→ BigQuery Sandbox
Solve 5 advanced SQL problems on LeetCode or DataLemur (analyst-specific practice)
→ DataLemur Free
